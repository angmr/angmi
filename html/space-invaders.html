Space Invaders

View Code on Github

A Space Invaders clone. Made using the Python programming language and turtle graphics, combined with the Pygame library for extra functionality. I modified the code specifically for the Windows operating system and I fixed all the bugs the original code had (there were many). In addition to that, I added more graphics that I created using Piskel, to enhance interactivity, I added sound effects, background music, and made extensive changes overall.
